# NDI TEMPLATE - Server Node

## Local Development

To install :
```
npm install
```

To start :
```
npm start
```

### Start as Docker
Note that it won't watch file changes, used for front development or hosting.

To start :
```
sudo ./scripts/start_docker.sh
```
